# `order` microservice

# Production

### Installation

```shell
docker build . --tag minos-ecommerce-microservice-ticket
```

### How to run?

```shell
docker run -it minos-ecommerce-microservice-ticket ...
```

## Development

### Installation

```shell
poetry install
```

### How to run?

```shell
poetry run microservice start
```
